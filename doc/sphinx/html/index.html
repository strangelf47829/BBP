Welcome to BBP’s documentation!

BBP is an extensible API in the form of libraries and conventions capable of hosting a minimal operating system in a portable manner. Currently, BBP is built on top of g++’s libc++, however BBP provides portability by hiding functions such as malloc() behind a garbage collector, so that the user ought not to call malloc() directly.

Although BBP is far from POSIX compliant, it does attempt to follow certain standards such as ECMA-35. All definitions of conventions created for this project can be found throughout this documentation, for which summaries can be found at (Stub).
